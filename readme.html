Aquarium Light

To use the WiFi Controller project for aquarium lights I slightly modified the wifi_controller main git branch and placed it in a new aquarium_light git branch. For aquarium lights each GPIO channel of the controller gets attached to a specific LED channel by an XY-MOS module. The hardware setup, schematic, customized webinterface and more are shown below.
