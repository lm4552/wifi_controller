Aquarium Light

To use the WiFi Controller project for aquarium lights I slightly modified the wifi_controller main git branch and placed it in a new aquarium_light git branch. For aquarium lights each GPIO channel of the controller gets attached to a specific LED channel by an XY-MOS module. The hardware setup, schematic, customized webinterface and more are shown below.

See project on GitHub

Webinterface

The webinterface allows to click or hover on a timeline graph to show the daily intervals and their purpose (e.g.Â dimming the light down, lunch break etc. ). By the indices shown in the graph it is possible to adjust the settings in the table below the graph. The brightness can be adjusted on a per LED channel basis to control the color temperature and spectrum.
